In this introductory level project, you'll use Sinatra to build a number guessing game.

Goals:

-Generate a random number on startup
-Have a web page where you can enter a guess
-Tell whether your guess was:
	-Way too high
	-Too high
	-Correct
	-Too low
	-Way too low
